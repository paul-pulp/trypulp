# PulpIQ - AI Operations Platform for Independent Coffee Shops

**Status:** Pre-Launch / Validation Phase  
**Launch Target:** 11 weeks from start  
**Current Phase:** Building landing page + pilot program

---

## Project Overview

PulpIQ is an AI-powered operations intelligence platform specifically designed for independent coffee shops (2-5 locations). We help cafés reduce waste by 20%, optimize labor costs, and increase customer retention through predictive analytics.

**Key Differentiator:** Coffee-shop-specific AI that works with existing POS systems (Square, Toast, Clover) at 60% lower cost than enterprise alternatives.

---

## Value Proposition

### For Coffee Shops:
- **Reduce waste 20-30%**: AI predicts optimal ordering for milk, pastries, beans
- **Optimize labor 10-15%**: Smart scheduling based on demand forecasting
- **Increase retention**: VIP tracking and churn prediction

### Pricing:
- **Free trial:** 6 weeks
- **Paid:** $199/month per location (vs $600+ for Toast + add-ons)

---

## Project Goals

### Phase 1: Validation (Weeks 1-11)
- [ ] Build landing page
- [ ] Get 20-30 form submissions
- [ ] Onboard 8-12 pilot cafés
- [ ] Deliver 6 weeks of weekly insights
- [ ] Achieve 2-4 paid conversions (25-33% rate)

### Success Criteria:
- ✅ 20+ form submissions = demand exists
- ✅ 10+ data uploads = qualified interest
- ✅ 2+ paid conversions = willingness to pay
- ❌ <10 submissions = kill project
